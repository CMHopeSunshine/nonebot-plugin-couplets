## 📖 介绍

基于[王斌给您对对联API](https://ai.binwang.me/couplet/)的Nonebot2对对联插件。

## 💿 安装

在 nonebot2 项目的根目录下打开命令行, 输入以下指令即可安装

```shell
nb plugin install nonebot-plugin-couplets
```


## ☀ ️指令
```
对联 <上联内容> (数量)
· 数量可选，默认为1
```
例如，`对联 苟利国家生死以 3`
